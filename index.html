학급 퀴즈 자동화 시작하기

Apps Script를 활용하여 퀴즈 채점과 피드백을 자동화하고 스마트하게 학급을 관리하세요.

퀴즈 자동화란?

구글 폼으로 수합된 학생들의 퀴즈 응답을 자동으로 채점하고, 오답 노트나 맞춤형 피드백을 학생 개별 이메일로 전송하는 등 교사의 반복적인 학급 관리 업무를 자바스크립트 기반의 Apps Script로 자동화하는 것을 의미합니다.

📝

자동 채점 시스템

학생들의 구글 폼 제출과 동시에 스프레드시트에서 정답을 비교하여 점수를 자동으로 계산합니다.

📧

맞춤형 피드백 메일

채점 결과와 함께 틀린 문제에 대한 해설 및 오답 노트를 학생 개별 이메일로 자동 발송합니다.

📊

성적 통계 분석

반별, 학생별 성적 분포와 문항별 정답률을 자동으로 분석하여 직관적인 대시보드를 구성합니다.

퀴즈 자동 채점 스크립트 예제

구글 폼 제출 시 자동으로 실행되어 점수를 계산하고 메일을 보내는 간단한 예제입니다.

function onQuizSubmit(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var row = e.range.getRow();
  var responses = e.values;
  
  // 간단한 채점 (예: 2, 3번 문항)
  var score = 0;
  if (responses[1] === "정답1") score += 50;
  if (responses[2] === "정답2") score += 50;
  
  // 점수 기록 및 메일 전송
  sheet.getRange(row, 4).setValue(score);
  MailApp.sendEmail(responses[3], "퀴즈 결과", score + "점 입니다.");
}
