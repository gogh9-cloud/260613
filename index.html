Google Apps Script 시작하기

업무 자동화의 강력한 도구, GAS를 활용하여 스마트하게 일하세요.

Apps Script란?

Google Workspace 기반의 경량 애플리케이션 개발 플랫폼입니다. 자바스크립트를 사용하여 Gmail, 스프레드시트, 드라이브 등 다양한 Google 서비스를 자동화하고 유기적으로 연동할 수 있습니다.

📊

스프레드시트 자동화

반복되는 데이터 입력, 정리 및 분석 작업을 자동화하여 업무 소요 시간을 획기적으로 단축하세요.

📅

일정 자동 등록

스프레드시트에 입력된 일정 데이터를 기반으로 구글 캘린더에 이벤트를 자동으로 생성합니다.

⚙️

사용자 정의 함수

기본 제공 함수로 해결하기 어려운 복잡한 계산식이나 로직을 직접 자바스크립트로 구현하여 사용하세요.

첫 번째 스크립트 작성하기

구글 캘린더에 새로운 일정을 생성하는 간단한 예제입니다.

function createCalendarEvent() {
  // 캘린더와 이벤트 정보 설정
  var calendar = CalendarApp.getDefaultCalendar();
  var title = "주간 업무 회의";
  var startTime = new Date('2026-06-15T10:00:00');
  var endTime = new Date('2026-06-15T11:00:00');
  
  // 이벤트 생성
  calendar.createEvent(title, startTime, endTime);
}
